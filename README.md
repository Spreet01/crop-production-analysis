Crop Production Analysis
 Project Overview

This project analyzes crop production data to uncover trends, patterns, and insights in agriculture.
It focuses on understanding production distribution across states, crop types, and time periods using data analysis and visualization.

Features
1. Crop-wise production analysis
2. State-wise comparison
3. Ear-wise trend analysis
   
Interactive visualizations
1. Identification of high & low production areas
